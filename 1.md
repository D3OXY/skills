# 1. Design Philosophy

This design language combines:

- Cinematic interface design
- Glassmorphism
- Spatial UI systems
- macOS-inspired layering
- Luxury enterprise software aesthetics
- Editorial minimalism
- Futuristic AI platform interfaces
- Floating operating-system-inspired panels

The UI should feel:

- intelligent
- atmospheric
- expensive
- responsive
- physically layered
- cinematic
- futuristic
- restrained

Not:

- playful
- colorful
- cartoonish
- aggressively animated
- flat
- noisy

The overall emotional tone:

> “High-end software crafted like luxury hardware.”

---

# 2. Core Visual Identity

## Foundation Principles

### A. Darkness as Material

Never use pure black.

Use:

```txt
#0A0A0B
#0E0F11
#101114
#15161A
#1B1D22
#202228
```

Darkness should feel:

- soft
- layered
- atmospheric
- cinematic

---

### B. Controlled Contrast

The UI relies on:

- low contrast surfaces
- high contrast typography
- sparse accent highlights

This creates:

- readability
- focus
- premium restraint

---

### C. Spatial Layering

Everything should feel detached from the background.

No flat interfaces.

Every surface:

- floats
- layers
- casts depth
- reacts to light

---

## Typography Colors

```css
--text-primary: rgba(255, 255, 255, 0.95);
--text-secondary: rgba(255, 255, 255, 0.72);
--text-muted: rgba(255, 255, 255, 0.45);
```

---

## Accent Philosophy

Accent colors should be:

- sparse
- meaningful
- intentional
- functional

Typical accents:

```txt
Electric Blue
Neon Orange
Soft Purple
Signal Green
Amber
Cyan
```

Avoid:

- rainbow palettes
- multiple competing accents
- saturated gradients everywhere
