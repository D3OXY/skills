# 34. Premium Feel Principles

## What Makes UI Feel Expensive

### Restraint

Premium interfaces avoid excess.

Use:

- fewer colors
- fewer accents
- cleaner spacing
- intentional motion

---

### Motion Hierarchy

Not every element should animate equally.

Important layers:

- hero motion
- active interactions
- focused components

Secondary layers:

- subtle atmospherics
- ambient lighting
- background drift

---

### Spatial Depth

Depth creates realism.

Use:

- shadows
- blur
- layered opacity
- perspective
- atmospheric gradients

---

### Typography

Editorial typography creates sophistication.

Characteristics:

- large headlines
- restrained weights
- clean spacing
- geometric structure

# 42. Advanced Motion Systems

## Floating Drift

Panels should drift subtly.

Example:

```txt
translateY oscillation
0px → 4px → 0px
```

Duration:

```txt
4s–12s
```

---

## Ambient Gradient Motion

Background gradients slowly shift.

Movement should be:

- nearly invisible
- atmospheric
- soft

---

## Blur Interpolation

Blur should animate during:

- entrances
- exits
- focus changes
- overlays

Example:

```txt

```

# 48. Performance Philosophy

## Performance Rules

Premium motion must remain smooth.

Prioritize:

- GPU accelerated transforms
- opacity animation
- transform animation

Avoid:

- layout thrashing
- excessive blur layers
- too many simultaneous particles

---

## Performance Friendly Properties

Preferred:

```css
transform
opacity
filter
```

Avoid animating:

```css
width
height
top
left
```
