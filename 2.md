# 21. Sidebar Design

## Sidebar Philosophy

The sidebar is not just navigation.

It behaves like:

- a control dock
- an operating system rail
- a spatial anchor
- a persistent command surface

The sidebar should feel:

- grounded
- atmospheric
- tactile
- elevated
- softly illuminated

---

## Sidebar Width Strategy

```txt
72px   → icon rail
220px  → compact sidebar
280px  → standard dashboard sidebar
320px  → enterprise systems
```

---

## Sidebar Surface Construction

```css
background: rgba(15, 15, 18, 0.82);
backdrop-filter: blur(28px);
border-right: 1px solid rgba(255, 255, 255, 0.04);
```

---

## Navigation Item Formula

```css
.nav-item {
    height: 44px;
    border-radius: 14px;
    padding-inline: 14px;
    display: flex;
    align-items: center;
    gap: 12px;
}
```

---

## Navigation States

### Idle

```css
color: rgba(255, 255, 255, 0.62);
```

---

### Hover

```css
background: rgba(255, 255, 255, 0.04);
transform: translateX(2px);
```

---

### Active

```css
background: rgba(255, 255, 255, 0.06);
border: 1px solid rgba(255, 255, 255, 0.08);
```

---

## Sidebar Entrance Animation

```txt
opacity: 0 → 1
translateX: -12px → 0
blur: 12px → 0
```

- cinematic depth

# 31. Hover Systems

## Hover Behavior

Hover should:

- improve illumination
- increase clarity
- deepen depth

- dramatically transform
- aggressively animate

glow intensifies
