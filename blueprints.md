# Composition Blueprints

Composition recipes for specific UI surfaces. Use alongside `cinematic-aesthetic.md` when building these surfaces in the cinematic style — or use the structures alone with any aesthetic.

## Composition Philosophy

The UI should feel layered like architecture:

```
Environment
↓
Atmosphere
↓
Glass Surfaces
↓
Panels
↓
Cards
↓
Interactive Controls
↓
Micro Feedback
```

### Composition Rules

- **Use large negative space** — creates luxury, focus, clarity, cinematic pacing
- **Avoid flatness** — always create depth, separation, atmosphere, motion hierarchy
- **Layer components** — never place everything on one plane; use overlapping cards, floating windows, shadow separation, blur depth

## Floating Command Palette Blueprint

Strongly inspired by Raycast, Spotlight, Arc.

### Structure

```
Search Input
↓
Command Results
↓
Preview Area
↓
Keyboard Shortcuts
```

### Styling — cinematic aesthetic

```css
background: rgba(15, 15, 18, 0.78);
backdrop-filter: blur(32px);
border-radius: 28px;
border: 1px solid rgba(255, 255, 255, 0.05);
```

### Animation

```
opacity: 0 → 1
scale: 0.96 → 1
blur: 12px → 0
translateY: 8px → 0
```

## Visual Atmosphere Recipe

Layered atmospheric background — five layers stacked for cinematic depth:

1. **Dark gradient base**
2. **Radial ambient glow**
3. **Noise texture overlay**
4. **Subtle vignette**
5. **Slow animated blur drift**

Result: cinematic atmosphere · depth · premium softness · immersive environment.

## Premium SaaS Patterns

Common patterns for premium dashboards/apps:

- **Floating analytics cards** — cards hover above background
- **Progressive dashboard reveal** — widgets animate sequentially
- **Ambient lighting** — panels softly illuminate
- **Cursor reactive highlights** — mouse movement affects lighting
- **AI streaming interfaces** — text streams progressively
- **Spotlight search** — floating command systems
